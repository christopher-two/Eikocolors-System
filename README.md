# Requerimientos

- Productos:
  - Nombre
  - Descripción
  - Precio
  - Categoría
  - Imagen del producto
  - id del producto (único)
- Cotizacion por Pdf.
- Catálogo de productos PDF:
    - Unico por cliente.
    - General por empresa.
- filtrado de busqueda por IA.
- distribuidores:
  - Nombre
  - Contacto
  - Teléfono
  - Email
  - Dirección
  - Productos que distribuye (relación con productos)
  - id del distribuidor (único)
